# 🚀 ShareKorbo – Campus Resource Sharing Platform

ShareKorbo is a mobile-first campus marketplace where students can **buy, sell, rent, and share resources** within a trusted university community.

---

## 📱 Features (Current Progress)

### 🔐 Authentication

* Login page
* Registration with student verification
* Verification status (Pending / Approved)

### 🛒 Marketplace

* Browse products/resources
* Responsive grid layout (mobile, tablet, desktop)
* Light/Dark mode toggle

### 📦 Product Details

* Full product view
* Owner information
* Booking/request system (UI ready)

### 📊 Activity & Requests

* Track requests and interactions (UI ready)

---

## 🛠️ Tech Stack

### Frontend

* React Native (Expo)
* NativeWind (Tailwind CSS for RN)
* React Navigation

### Backend (Planned / In Progress)

* Node.js / Express OR Python (FastAPI)
* MySQL Database

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

```
git clone https://github.com/mdrifat-hossain/ShareKorbo.git
cd ShareKorbo
```

---

### 2️⃣ Setup Frontend

```
cd frontend
npm install
npx expo start
```

---

### 3️⃣ Setup Backend (if applicable)

```
cd backend
pip install -r requirements.txt
```

---

## 🔒 Environment Variables

Create a `.env` file if needed:
